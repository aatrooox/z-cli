# z-cli 功能概览

## 项目定位

z-cli 是一个**效率类命令行工具**，专为解决工作中和个人开发过程中的重复或繁琐问题而设计。

## 设计理念

- **一体化工具箱** - 集成多个常用工具，避免频繁切换
- **自动化流程** - 减少手工操作，提升工作效率
- **可配置** - 支持全局配置管理，适应不同使用场景

## 核心功能模块

### 图片处理工具

**问题场景**：图片文件过大影响加载速度，需要批量压缩优化。

**解决方案**：
- 基于 Sharp 的高质量图片压缩
- 支持多种图片格式（PNG、JPG、GIF、WebP等）
- 支持批量处理和条件过滤
- 可自定义压缩质量和输出格式

**适用场景**：
- 网站图片优化
- 博客写作
- 图片资源批量处理

### 配置管理

**问题场景**：多台设备或团队协作需要同步配置。

**解决方案**：
- 配置文件存储在用户目录（`~/.zzclub-z-cli/config.json`）
- 支持导入/导出配置
- 支持快速设置关键配置项

## 技术特点

- **ES Modules** - 使用现代 JavaScript 模块系统
- **跨平台** - 支持 Windows、macOS、Linux
- **多运行时** - 支持 Node.js 和 Bun
- **友好提示** - 使用 ora 和 chalk 提供清晰的终端反馈

## 典型工作流

### 场景：批量图片处理

```bash
# 1. 批量压缩图片
zz tiny -d ./images -q 80

# 2. 条件过滤压缩
zz tiny -d ./images -co thumbnail -q 75

# 3. 格式转换
zz tiny -f ./image.png -t webp -q 80
```

## 限制与注意事项

### 图片压缩

- 支持的格式：PNG、JPG、JPEG、GIF、WebP
- 压缩质量范围：1-100
- GIF 颜色保留范围：2-256

## 扩展性

项目采用命令注册模式，添加新功能只需：

1. 在 `src/command/` 下创建新命令文件
2. 导出符合规范的命令对象
3. 在 `src/index.js` 中注册命令

欢迎提出新需求或贡献代码！
